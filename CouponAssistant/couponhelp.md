### 如何得到京东或者拼多多的商品链接？

#### 1、京东APP

只需要打开对应商品页面，在右上角找到分享图标，点击，然后再点击复制链接

京东APP链接通常如：

```
【京东】https://3.cn/2Dp-Jq9p?jkl=@ZEF6R4zVDkTp@ ZH9112 「嘉顿有福气饼干礼盒1575g」
点击链接直接打开 或者复制文案打开京东
```

![](C:\Users\maiwu\Desktop\640.png)

#### 2、拼多多APP

和京东APP类似，也是右上角分享图标，然后找到复制链接，拼多多复制按钮需要往左滑动才可以找到。

拼多多目前只支持APP访问。

拼多多APP链接通常如：

```
https://mobile.yangkeduo.com/goods.html?ps=tQETM6eZrL
```

![](C:\Users\maiwu\Desktop\640 (1).png)

#### 3、京东网页

京东网页，打开商品页面，再在浏览器地址栏中复制链接就可以。

京东网页链接通常如：您也可以直接复制地址中的id：1441077，也是可以。

```
https://item.jd.com/1441077.html
```

![](C:\Users\maiwu\Desktop\640 (2).png)

## 如何获取优惠券？

获取到链接后，你只需要打开我们的应用，拷贝链接到文本框，点击”查询优惠券“按钮，就可以获得优惠券信息。
